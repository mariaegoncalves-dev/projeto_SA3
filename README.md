# SISTEMA DE ESTOQUE SIMPLIFICADO

DESCRIÇÃO: Um sistema de linha de comando simples e interativo desenvolvido em Python para gerenciar o estoque de ferramentas e materiais.
O projeto permite cadastrar, listar, buscar e atualizar produtos de forma rápida através de uma interface de menu estruturada.

FUNCIONALIDADES:
1. Adicionar Produto: Permite cadastrar novos itens informando ID, nome, quantidade e posição.
2. Listar Produtos: Exibe todos os produtos atualmente cadastrados no estoque.
3. Buscar por ID: Procura e localiza um produto específico utilizando seu número de identificação única.
4. Atualizar Produto: Localiza um produto pelo ID e permite alterar a sua quantidade em estoque.
5. Remover Produto: Ele remove um produto específico do estoque utilizando seu ID
6. Sair: Encerra a execução do programa de forma segura.
